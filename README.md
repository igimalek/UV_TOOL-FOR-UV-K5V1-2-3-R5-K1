# UV_TOOL UNREAL
Универсальный флешер работающий под Win7-11,редактор каналов и бендов для всех версий UV-K5 (V1/V2/V3) UV-R+ (с тайп си и без) UV-К1.
На борту встроенный hex для конфига и калибровок, редактор каналов, автоматическое восстановление до стока для всех раций.
Защита от выбора неверной прошивки. Три уровня проверки соответствия железа и файлов. Вы не сможете прошить не ту прошивку на не ту версию.
Редактор бендов поддерживает прошивки OURO КА-52 и Robzyl K1.
Для загрузки бендов и каналов и их редактирования в прошивке КА-52, в редакторе каналов перед загрузкой из рации или импорта образа нажать кнопку "OURO MODE"
По кнопке "БЕНДЫ" - откроет окно бендов для КА-52 и Robzyl K1. 
Для быстрой загрузки готового образа используйте кнопку импорта и файл KA-52 К1_BANDS.img из ветки с прошивкой.

Universal flasher running under Win7-11, channel and band editor for all versions of UV-K5 (V1/V2/V3) UV-R+ (with and without type C) UV-K1.
On board built-in hex for config and calibrations, channel editor, automatic restore to stock for all walkie-talkies.
Protection against selecting the wrong firmware. Three levels of verification of the compliance of the hardware and files. You will not be able to flash the wrong firmware on the wrong version.
The band editor supports OURO KA-52 and Robzyl K1 firmware.
